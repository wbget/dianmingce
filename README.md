# 点名册

![qrcode](https://user-images.githubusercontent.com/12776391/75139290-34d5f680-5727-11ea-8fd0-a7f3ddd3ef20.png)

使用 Remax + Mobx + VantUI 开发的微信小程序。

## 开始开发

安装依赖

```bash
$ npm install
```

开始构建

```bash
$ npm run dev
```

使用微信小程序开发者工具打开项目下的 `dist` 目录。

## 发布

```bash
$ npm run build
```

使用微信小程序开发者工具上传版本。
